# BigdataStudy
big data 분석에 필요한 기본적인 라이브러리와 파이썬 예제

#### BeautifulSoup
> HTML과 XML을 분석해주는 Library  

#### Selenium
> 웹 브라우저를 원격 조작할 때 사용하는 도구  
  (url을 열고 클릭, 스크롤, 문자 입력, 스크린샷 등의 조작 가능)
#### PhantomJS
> 화면없이 명령줄에서 사용할 수 있는 웹 브라우저

#### scikit-learn
> 머신러닝 프레임워크

#### Pandas
> python 데이터 분석, 처리에 특화된 라이브러리
